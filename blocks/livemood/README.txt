Live-Mood block
for Moodle
Author: Franck Chionna (infos@live-school.net)

NOTE:
Live-Mood block all versions (1.5 to 3.x) download are also available from https://secure.live-school.net/indexOrg.lol

To install:
1. Copy and unzip the livemood-X.X.X.zip folder to your moodle blocks folder.
2. Visit the Notifications page (Site Administration -> Notifications)
3. Click install Live-Mood block plugin
4. Register a Manager secret key by clicking on the link "Get my Admin/Manager secret key"
5. Add a block_livemood on your front page or whever you wish
6. Contact me if you need support
7. You are ready to Live-Mood!
